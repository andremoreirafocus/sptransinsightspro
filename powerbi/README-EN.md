# SPTrans Insights Dashboard (Power BI)

This directory contains the Power BI report used to visualize the data processed by the pipeline.
The dashboard queries the tables generated by the data-refinement processes through DirectQuery.

## Dashboard content and data

The dashboard consumes data from the refined layer and presents metrics and KPIs generated by the project’s processing components, as described below.

1. **Trip monitoring**
   Visualizes the start time, end time, and duration of trips completed in the last few hours.
   Also shows average trip duration in recent hours by period of day.
   Filtering is available by trip, identified by route and direction (`0` or `1`), and optionally by vehicle.
   Built from the finished-trips data generated for the refined layer.

2. **Real-time tracking**
   **Position map**: identifies current bus positions at query time.
   Filtering is available by trip, identified by route and direction (`0` or `1`), and optionally by vehicle.
   Built from the latest-position data generated for the refined layer.

## How to use

1. Make sure the PostgreSQL tables have already been created and populated by the project’s refinement processes.
2. Open [sptrans-dashboard.pbix](./sptrans-dashboard.pbix) in Power BI Desktop.
3. Update the connection credentials so they point to your local PostgreSQL container.
